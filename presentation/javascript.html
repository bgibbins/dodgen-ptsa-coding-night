HTML, CSS & Javascript

Introduction to how they work together to power web and mobile apps.

Javascipt provides function to the structure and presentation of the application. The original "language" was written in 10 days by an employee of Netscape (now Mozilla).

1996-1997 was an exciting time for web developers because Javascript was the first language that actually operated in the browser itself.

This meant that the application could run faster since it didn't need to contact a remote server over a sometimes slow Internet connection. This was before broadband Internet connections were popular in homes and most people were using a modem to "dial up to the ISP" and "get on the Internet".

Now we carry our connection to the Internet in our pocket and connect to wi-fi when we can and we can use Javascript to change things on the page, even on our mobile devices.

Now let's move on to a simple application.

A User Story

As a teacher I would like to be able to keep track of each student by where they sit at their desks in each classroom.

Requirements:

  • 3 total class rooms
  • Class rooms all have hold 12 desks
  • Desks are arranged in 3 rows and 4 columns

Application will use:

  • HTML to designate the structure of the data, then
  • CSS to present the structure and finally,
  • Javascript to manage the data for students in each classroom
