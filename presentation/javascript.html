HTML, CSS & Javascript

Introduction to how they work together to power web and mobile apps.

Javascipt provides function to the structure and presentation of the application. The original "language" was written in 10 days by an employee of Netscape (now Mozilla).

Things were moving at a very quick pace. 1996-1997 was an exciting time for web developers because Javascript was the first language that actually operated in the browser itself. Many of us eagerly awaited Internet Explorer as one of the early adopters of Javascript in the browser.

This meant that the application could run faster since it didn't need to contact a remote server over a sometimes slow Internet connection. Remember, this was before broadband Internet connections were popular in homes and most people were using a modem to "dial up to the ISP" and "get on the Internet".

Now we carry our connection to the Internet in our pocket and we can use Javascript to change things on the page, even on our mobile devices.
