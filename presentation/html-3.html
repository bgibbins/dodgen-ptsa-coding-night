Bloating

More and more and more custom tags came along...

...with tags and presentation differing between the companies entering the browser market.

  • The ability to add color to individual sections of text was added using a very flaky "font" tag.
  • and blinking text (yes, it was really a thing)
  • and developers wanted to add functionality and the ability to interact with the HTML with "real" programs
  • and designers wanted to make the presentation better by adding color and drop-shadows and animation
  • and still more specialized tags were added
  • and so began the browser wars
  • Before too long, the original purpose of HTML - the structure, was getting bloated with custom tags that affected the layout on the screen.

Something had to be done...
