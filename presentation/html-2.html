HTML, CSS & Javascript

Introduction to how they work together to power web and mobile apps.

Images

With the formal specification in 1993 came the ability to add images. The GIF (or Graphics Interchange Format) had been introduced in 1987 by CompuServe. So developers of the HTML standard added a new tag to make them visible in the broswer.

But, GIFs only have a maximimum of 256 individual colors, so along came JPEG (or JPG) with a the potential of 16 million different colors.

Bloating

Then more and more custom tags came along...

  • The ability to add color to individual sections of text was added using a very flaky "font" tag.
  • and blinking text (it was really a thing)
  • and more specilized tags were added
  • Before too long, the original purpose of HTML, the structure, was getting bloated with custom tags that affected the layout on the screen.

Something had to be done.
