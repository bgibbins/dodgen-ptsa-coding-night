HTML, CSS & Javascript

Introduction to how they work together to power web and mobile apps.

HTML (Hyper Text Markup Language) provides the structure in this important group of three technologies and was the first to be introduced. Originally published in a paper by Tim Berners-Lee in 1991 in a document called "HTML Tags", then formally in 1993 as as a specification "Hypertext Markup Language (HTML)" Internet-Draft". The initial abilities of HTML were very limited. As developers we only had 18 tags in the MARKUP LANGUAGE used to describe what would be seen in what we have come to know today as browsers.

The simplified purpose of HTML was to create a standard markup language which could be used with all computer systems to display content in the same way and allow each computer (server) to be linked together such that a user could click a link in the text to get to the next related document.

Simply defined a markup language is one that uses very specific text wrapped around other text that makes the text between two points behave in a predefined manner.
