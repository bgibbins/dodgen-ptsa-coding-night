A User Story

"As a teacher I would like to be able to keep track of each student by where they sit at their desks in each classroom."

Requirements:

  • 3 total class rooms
  • Class rooms all have 12 desks
  • Desks are arranged in 3 rows and 4 columns

Application will use:

  • HTML to designate the structure of the data, then
  • CSS to present the structure and finally,
  • Javascript to manage the data for students in each classroom
